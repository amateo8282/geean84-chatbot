# 기안84 챗봇 디자인 시스템 (Kian84 Design System)

이 문서는 기안84의 웹툰 '복학왕' 스타일을 기반으로 한 챗봇의 디자인 원칙과 가이드를 정의합니다.

## 1. 디자인 원칙 (Design Principles)

- **거친 사실주의**: 정교하고 깔끔한 라인보다는 손으로 그린 듯한 거칠고 불규칙한 선을 지향합니다.
- **빈티지한 색감**: 약간 바랜 듯한 노란색, 회색조의 베이지 등 복학왕 특유의 생활감이 느껴지는 색상을 사용합니다.
- **유머와 해학**: 사용자 인터페이스 곳곳에 기안84 특유의 위트 있는 요소(말풍선, 효과음 등)를 배치합니다.

## 2. 컬러 팔레트 (Color Palette)

| 구분 | 색상 코드 | 용도 |
| :--- | :--- | :--- |
| **Background** | `#F5F5F0` | 전체 배경 (약간 바랜 종이 느낌) |
| **Primary** | `#333333` | 텍스트 및 주요 테두리 (진한 먹색) |
| **Accent (User)** | `#E8E4D9` | 사용자 말풍선 배경 |
| **Accent (Bot)** | `#FFFFFF` | 챗봇 말풍선 배경 |
| **Point** | `#FFD700` | 강조 요소 (노란색 계열) |

## 3. 타이포그래피 (Typography)

- **기본 폰트**: `Nanum Pen Script` (손글씨 느낌 강조)
- **보조 폰트**: `Pretendard` 또는 시스템 기본 폰트 (가독성이 필요한 긴 텍스트)

## 4. 컴포넌트 스타일 (Component Styles)

### 4.1. 테두리 (Borders)
- 모든 컨테이너는 `DrawnBorder`를 사용하여 SVG 필터(`feTurbulence`)가 적용된 불규칙한 선을 가집니다.
- 테두리 두께는 2px~3px를 권장합니다.

### 4.2. 말풍선 (Speech Bubbles)
- 웹툰의 말풍선 형태를 따르며, 꼬리 부분이 손으로 그린 듯한 곡선을 가집니다.
- 그림자는 `2px 2px 0px rgba(0,0,0,0.1)`와 같이 플랫한 형태를 사용합니다.

## 5. 이미지 가이드 (Image Guide)

- **캐릭터**: 기안84의 '우기명' 스타일의 2D 일러스트.
- **배경**: 원룸, 대학 강의실 등 웹툰의 주요 배경을 흐릿하게 처리하여 사용.
- **도구**: 이미지 생성 및 편집에는 `나노바나나프로`를 사용합니다.
